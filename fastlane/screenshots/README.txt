iOS Screenshots Directory

This directory contains screenshots for App Store Connect.

Structure:
- en-US/ - English (US) screenshots
- Add more language folders as needed (e.g., es-ES/, fr-FR/, de-DE/)

Each language folder should contain screenshots organized by device size.

To generate screenshots automatically:
1. Set up fastlane snapshot
2. Run: fastlane snapshot
3. Screenshots will be generated in this directory

Manual screenshot capture:
- Use iOS Simulator or real devices
- Capture at actual device resolution
- Save as PNG files
- Organize by language and device size
